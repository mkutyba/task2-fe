# Task2Fe
[![Build Status](https://travis-ci.org/mkutyba/task2-fe.svg?branch=master)](https://travis-ci.org/mkutyba/task2-fe)

Run in docker:
```
git clone https://github.com/mkutyba/task2-fe.git
cd task2-fe
docker-compose run node npm install
docker-compose up -d
```

or in your system:
```
git clone https://github.com/mkutyba/task2-fe.git
cd task2-fe
npm install
npm start
```
